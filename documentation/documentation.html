Images


Images can be modificated from the html document.

For example : If we want to change the images from the mosaic,we go to our index.html (home-page) and look for <!-- START MOSAIC -->

index.html 
          
<div class="mosaic-item x2">
  <a class="section-background bg" style="background-image: url(img/ournewimage.jpg)">
</div>

Desktop-menu image

If you are going to add or remove desktop-menu links maybe your menu images will not be well-adjusted.You need to play with background-position of (1) and (2) for a perfect fit.

index.html 
          
<li class="men list-menu" style="background-image:url(img/menu.jpg)"> (1)
  <a class="first-level" href="">MEN</a>
  <div class="second-level">
      <div class="image-container" style="background-image:url(img/menu.jpg)"> (2)
      ...
      </div>
      ...
  </div>
</li>
styles.css 
          
(1)
.men:hover {
    background-position: calc(-221px + 0px) -1px;
}
          
          
          
(2)
.menu-nav .men .second-level .image-container {

    background-position: 0 -40px;

}

You can use this image as a guide.
